官方demo演示网址：http://jasonload.zglingjie.com/demo/
publish文件夹为jasonload框架发布包，里面包含模块化页面js模板template.js和页面调用模板index.html。

publish发布包文件结构说明：
1、assets文件夹为所有静态文件根目录
  1.1assets/jasonload文件夹为jasonload框架文件目录，可以把第三方组件库文件夹放在这里，类似jasonwap手机端控件库一样部署
  1.2assets/js文件夹存放站点js文件，也可以自己重新定义文件夹名称

请先打开assets/js/config.js配置站点根目录（即index.html所有目录）site_url的值后，再运行index.html即可查看运行效果

文件夹doc下面为框架的API文档。其中：
1、jasonwap子文件下所有文档为手机端控件的API文档。
